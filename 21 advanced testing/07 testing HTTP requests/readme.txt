Протестируйте функцию getUserMainLanguage(username, client), которая определяет язык на котором пользователь
создал больше всего репозиториев. Для реализации этой задачи, функция getUserMainLanguage выполняет запрос через
@octokit/rest, который извлекает все репозитории указанного пользователя (по первому параметру username).
Каждый репозиторий в этом списке, содержит указание основного языка репозитория. Эта информация используется для поиска
того языка, которые используется чаще.
